IBM Watson Platform Software Development Kit: Quick Start Guide

Overview

Welcome to the IBM Watson™ Platform SDK for Eclipse. The SDK provides a comprehensive set of applications and libraries to enable you to develop J2EE and JavaScript applications for Watson. Additionally, a PHP example is included as a preview.

For more documentation, including complete installation and configuration instructions, see the IBM Watson Platform SDK Developer's Guide in the com.ibm.watson.sdk.doc project.

Step 1: Installing and configuring the SDK

You install the Watson Platform software development kit (SDK) in Eclipse by importing the SDK project workspace archive file.

Installing the SDK

  1. In Eclipse, select File > Import.
  2. Select General > Existing Projects into Workspace
  3. Select the archive file option and browse to the location where you saved the SDK.

Associating SDK projects with the server

To run the web-based applications and samples, follow these steps to deploy the Eclipse web projects to a web application server:

  1. Install a web application server, such as Tomcat, in your Eclipse environment.
  2. In the Servers view in Eclipse, right-click the server and select Add and Remove Projects.
  3. In the list of available projects, select the following projects and click Add, and then click Finish.
    • com.ibm.watson.sdk.dojo
    • com.ibm.watson.sdk.wea.toolkit
    • com.ibm.watson.sdk.wea.toolkit.samples
    • com.ibm.watson.sdk.wea.toolkit.app.samples
    • com.ibm.watson.sdk.wea.proxy
  4. Publish the applications by right-clicking the server and selecting Publish.
  5. If necessary, start the server.

Step 2: Ask a question

The IBM Watson Platform SDK includes an end-to-end sample application that uses the JavaScript widgets that are included in the SDK. The application posts a question to Watson and returns the answer.

Access the application at http://localhost:8080/watson/qa/demo/index.html. The link assumes that your web browser runs on localhost on port 8080.

Step 3: Explore the code

Explore either the Java code or the PHP library.

Sample Java code to call the Question and Answer service

The SDK includes several samples that are written in Java™ to call the Question and Answer REST service. Both the code and the output of the samples might give you ideas for how to code your own application. The samples are stored in the com.ibm.watson.sdk.qaapi.samples project under the com.ibm.watson.sdk.qaapi.web.rest.questionanswer package:
  • QAJsonSample: Asynchronous implementation that returns JSON data.
  • QASyncJsonSample: Synchronous implementation that returns JSON data.
  • QASyncXmlSample: Synchronous implementation that returns XML data.
  • QAXmlSample: Asynchronous implementation that returns XML data.

Run a sample in Eclipse by right-clicking the package and selecting Run As > Java Application.

PHP sample

The SDK includes a PHP library that you can use to call services for Watson. You need a PHP runtime environment that is compiled with cURL support and connected to a web server.

Copy the contents of the com.ibm.watson.sdk/PHP/ directory to your web server

To run the sample, go to http://localhost/question.html. The link assumes that your SDK PHP library runs on localhost.

Step 4: Explore the JavaScript samples

The SDK provides sample files and widgets to help you create rich, web-based applications that are enabled for Watson.

View the following samples in your web browser. The links assume that your web browser runs on localhost on port 8080.

  • Confidence Bar
  • Evidence Button
  • Evidence Viewer
  • jQuery example
  • Progress Bar example
  • Progress Dialog example
  • Question Input Dialog
  • Question Input
  • Rating example
The HTML files are stored in the com.ibm.watson.sdk.wea.toolkit.samples project in the WebContent directory.
