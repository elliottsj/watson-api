Watson Platform Software Development Kit

Watson Enabled Application (WEA) JavaScript Toolkit Widgets

Overview

The Watson Enabled Application JavaScript toolkit provides a JavaScript files and widgets to create rich Web-based applications that can be used in creating your own applications.

The SDK provides three projects to help you with create rich web applications using JavaScript:

  1. com.ibm.watson.sdk.dojo

    The com.ibm.watson.sdk.dojo project contains a version of Dojo 1.7 which is required for use by WEA Toolkit. The Dojo toolkit can be used to develop other JavaScript based applications and is not specific to WEA Toolkit.

  2. com.ibm.watson.sdk.wea.toolkit

    The com.ibm.watson.sdk.wea.toolkit contains the base widgets which are used by the WEA Toolkit and can be embedded and used with your applications.

  3. com.ibm.watson.sdk.wea.toolkit.samples

    The com.ibm.watson.sdk.wea.toolkit.samples project contains sample applications which shows uses of the WEA Toolkit widgets and can act as starting point for creating your own web applications.

Installation and Configuration

The three projects, com.ibm.watson.sdk.dojo, com.ibm.watson.sdk.wea.toolkit, and com.ibm.watson.sdk.wea.toolkit.samples exist as Eclipse web projects and need to be deployed to a web application server. Start by configuring a Web Application Server such as Tomcat v7 in your Eclipse environment and deploying the three web projects to your server.
